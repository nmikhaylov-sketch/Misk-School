Do not apply code changes directly to files unless the user explicitly asks to make the change. Instead, provide suggestions and code snippets the user can review and apply themselves.

**Why:** User wants to stay in control of what goes into the files and review changes before they are applied.

**How to apply:** When asked "how do I", "how can I", or "any suggestions", respond with explanation and snippet only. Only use Edit/Write tools when the user says something like "apply this", "make the change", "update the file", etc.
